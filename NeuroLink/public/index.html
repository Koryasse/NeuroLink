Ouvrir NeuroLink

Un espace à nous, rien qu’à nous


Ce chat n’est pas public, il est réservé uniquement aux gens de notre classe. Pas de spam, pas de pub, pas d'inconnus. Juste nous, avec un espace où on peut échanger rapidement, se filer des infos, s’entraider ou juste papoter. Il est là quand t’en as besoin, sans distraire quand t’en as pas envie.

Un chat qui s’adapte à ton style


Envie d’un thème sombre pour moins te cramer les yeux ? Ou peut-être d’un camouflage discret pour que le prof pense que tu prends des notes ? T’inquiète, on a prévu le coup. Change l’apparence du chat en un clic, choisis ton ambiance, et active le mode “furtif” pour que tout ressemble à une prise de notes ou un site normal. Pratique quand tu veux rester connecté sans te faire griller.

Échange de fichiers, messages et même du code


Envoie un PDF, colle un bout de code ou partage une image — tout passe facilement. Que tu bosses sur un projet de groupe ou que t’aies juste besoin d’un retour rapide, le chat est là pour centraliser les échanges, sans prise de tête.

Vous ne pouvez plus faire défiler.
Aller mieux discuter.
